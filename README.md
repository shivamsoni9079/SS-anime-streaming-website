# SS-anime-streaming-website
SS!Anime is a responsive anime streaming and discovery website built using HTML, CSS, and JavaScript.  
The project uses the Jikan API to fetch real-time anime data including trending anime, top airing shows, genres, popular series, and detailed watch pages.

The website is designed with a modern anime-style UI, responsive layouts, dynamic anime sections, search functionality, pagination, genre filtering, mobile navigation, and localStorage-based login UI.

---

## Features

- Responsive Anime Website UI
- Trending Anime Section
- Top Airing Anime
- Most Popular Anime
- Genre Filtering
- Anime Search System
- Dynamic Anime Watch Page
- Pagination System
- Mobile Search Bar
- Sidebar Navigation Menu
- Login UI using localStorage
- Dynamic Navbar Loading
- Anime Cards with Episode Info
- Duplicate Anime Filtering
- Real-time Anime Data using Jikan API

## Main Functionalities

### Anime Search
Users can search anime by title from both desktop and mobile search bars.

### Genre Filtering
Selecting a genre dynamically loads anime related to that genre.

### Anime Watch Page
Each anime card redirects to a dedicated watch/details page.

### Pagination
Anime pages support dynamic pagination for browsing multiple pages.

### Login UI
Simple login system using browser localStorage.
